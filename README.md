# Salon Mate (미용실 예약 관리)

미용실 원장님을 위한 1인 예약 관리 PWA(Progressive Web App)입니다.
모바일과 데스크탑에서 앱처럼 설치하여 사용할 수 있으며, Firebase를 통해 데이터를 실시간으로 동기화합니다.

## ✨ 주요 기능

- **캘린더 뷰**: 월별 예약 현황을 직관적으로 확인
- **예약 관리**: 예약 추가, 수정, 삭제 (CRUD)
- **고객 관리**: 고객별 예약 히스토리 검색 및 조회
- **PWA 지원**: 앱 설치, 오프라인 지원 (Service Worker), 홈 화면 아이콘
- **알림 위젯**: '남은 예약', '예약하기' 등 직관적인 대시보드 위젯

## 🛠 기술 스택

- **Frontend**: React (ES Modules via importmap), Tailwind CSS
- **Backend**: Firebase (Firestore)
- **Deployment**: Static HTML/JS hosting

## 🚀 실행 방법

이 프로젝트는 별도의 빌드 과정 없이 최신 브라우저에서 바로 실행됩니다.

1. 프로젝트 폴더를 VS Code 등으로 엽니다.
2. `Live Server` 확장 프로그램을 사용하거나, 로컬 웹 서버를 통해 `index.html`을 실행합니다.

## 📱 앱 설치 방법

1. 브라우저(Chrome, Safari 등)로 웹사이트에 접속합니다.
2. 주소창의 '설치' 아이콘을 클릭하거나, 공유 메뉴의 '홈 화면에 추가'를 선택합니다.
